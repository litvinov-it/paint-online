
# Paint online
stack: `react`, `mobx`, `websocket`, `nodeJS`
## start project

1. git clone

```bash
  git clone https://github.com/litvinov-it/paint-online.git
```

2. start backend
```bash
    cd ./server
    npm run serve
```

3. start frontend
```bash
    cd ./client
    npm run start
```

4. Open site http://localhost:3000/
5. Write name
6. Create room and connect room from id
